Este es el repositorio del desarrollo de la aplicación de ComiCommerce.

Para ejecutar la app puedes descargar el repositorio y abrir el proyecto en la última versión de Android Studio o descargar el archivo .apk desde el siguiente enlace:
https://drive.google.com/file/d/1pr1b8evOMOAk7-lyqU6TP3D3aSM4Q8oz/view?usp=sharing

Este proyecto implementa la siguiente API, creada para realizar un backend funcional:
https://github.com/gonzaloaliaga/API-mongodb
